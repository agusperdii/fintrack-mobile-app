## Detailed Flow Scenarios

### 1. Daily Engagement Flow
1. **Entry**: User launches app.
2. **Action**: Daily Check-in Card appears or user clicks Streak Button in AppBar.
3. **Reward**: Streak count increments, visual feedback given.
4. **Reflection**: User clicks Streak Button to view "My Streak" summary and progress.

### 2. Smart Logging Flow (OCR)
1. **Trigger**: User receives a paper receipt.
2. **Action**: Clicks "Scan" in Quick Actions.
3. **Process**: App captures image -> OCR extracts amount and title.
4. **Validation**: User confirms/edits extracted data.
5. **Completion**: Transaction saved, balance updated instantly.

### 3. Budgeting & Alerting Flow
1. **Setup**: User goes to Spending Target List -> Add Target.
2. **Action**: User sets Rp 2.000.000 for "Food" this month.
3. **Monitoring**: Every "Food" expense reduces the remaining budget.
4. **Notification**: System sends a "Nudge" if 80% of budget is reached.
5. **Review**: User checks "Analisa" to see which category eats up most of the budget.

### 4. Periodic Review Flow
1. **Visual**: User looks at "Wawasan Mingguan" (Weekly Pulse) on Dashboard.
2. **Drill-down**: User navigates to "Analisa Page" for detailed charts.
3. **Comparison**: User checks "Monthly Summary" to compare this month vs last month.
4. **Adjustment**: User modifies Spending Targets for the next period.
